cppemacs::cell Struct Reference

#include <cppemacs/core.hpp>

Description

General Emacs value wrapper.

A cell is like a simple value, but additionally carries around the envw it is part of. This provides a few benefits:

  • Cells can be called as functions, performing type conversions on the arguments.
  • operator==() and operator bool() can be used to idiomatically check (identity) equality and nil-ness.
  • They can be extracted to C++ values easily.
envw env = ...;
value val = ...;
// envw::inject()/->* return a cell
cell cel = env->*val;
// cells can be called and perform conversions
cel(1, 2, 3);
// code should check for errors
env.maybe_non_local_exit();
// cells make it easy to call Emacs functions
(cel->*"message")("Hello!"_Estr);
