cppemacs::literals::estring_literal Struct Reference

#include <cppemacs/literals.hpp>

Description

A reference to a string, can be created with ""_Estr().

This is a barebones version of std::string_view, and converts to an Emacs string.

envw env = ...;
using namespace cppemacs::literals;
value str = env->*"This is a string!"_Estr;
