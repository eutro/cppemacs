Description

template<typename F>
struct cppemacs::module_function< F >

A wrapper over F that allows it to be converted to an Emacs function.

A reference to F must be invocable with (emacs_env *env, ptrdiff_t nargs, value *args), returning a value. This differs from a simple emacs_function, in that the data parameter is converted to F according to the data_repr, instead of being passed to F.

F will be wrapped in envw::run_catching(), and so is free to throw exceptions.

See also
make_module_function() and make_spreader_function() for creating instances.

Member Typedef Documentation

◆ data_repr

template<typename F >
using cppemacs::module_function< F >::data_repr = module_function_repr<F>

The representation used to convert between the data pointer and the stored instance of F.

Unless module_function_repr is (further) specialized by the user, the behaviour is as follows:

  • If F is trivially copyable and destructible, and it is smaller than void*, it is directly stored in data.
  • Otherwise, F is move-constructed into a heap-allocated pointer, and a finalizer is registered for the Emacs function which default-deletes it, silently ignoring any exceptions in the finalizer.
