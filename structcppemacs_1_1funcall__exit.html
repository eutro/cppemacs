cppemacs::funcall_exit Struct Reference

#include <cppemacs/core.hpp>

Description

Enum wrapper for emacs_funcall_exit.

When converted to bool, this is true if there is a signal or throw pending.

See also
envw::non_local_exit_check() and envw::non_local_exit_get() for obtaining the current function exit status.

See the \manlink{manual|Module-Nonlocal.html}.
