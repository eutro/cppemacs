Convert a user_ptr from Emacs, with the GC still responsible for the object.

This function type-checks the argument by comparing val's registered finalizer to fin().

Postcondition
The resulting user_ptr is not owned by us: it must not be deleted, nor converted back to an Emacs value.

Convert a user_ptr to Emacs, with the GC becoming responsible for the object.

This function is safe against pending non-local exits, in that if the value cannot successfully be passed to Emacs' garbage collector, it will be deallocated immediately.

Precondition
The pointer must be owned by us: it has not been converted before, nor was it obtained from an existing value.
