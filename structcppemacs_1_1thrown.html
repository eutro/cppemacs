cppemacs::thrown Struct Reference

#include <cppemacs/core.hpp>

Description

An exception representing an Emacs throw exit.

Warning
This should not be constructed or thrown if there is already a pending non-local exit, as Emacs module API functions will not return meaningful values.
See also
envw::run_catching(), which throws in Emacs if this is caught.

envw::rethrow_non_local_exit(), which throws this if an Emacs throw is pending.
