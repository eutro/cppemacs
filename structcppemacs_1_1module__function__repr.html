template<typename F, typename = void>
struct cppemacs::module_function_repr< F, typename >

Data representation for storing C++ functions in Emacs closures.

Emacs lets us store one pointer worth of data along with a module function.

The default implementation puts it in an off-heap pointer and attaches a finalizer. There is a specialization that puts F directly as the function data instead (if it fits).
