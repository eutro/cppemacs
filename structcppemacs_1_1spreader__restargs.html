cppemacs::spreader_restargs Struct Reference

#include <cppemacs/utils.hpp>

Description

Used to provide a span-like argument to a spreader function .

This is just an pointer to arguments and a length. This can implicitly be converted via a (value *begin, value *end) constructor, such as std::vector. In C++20, this is also compatible with the ranges library, and can be converted to a std::span<value>.

env->*make_spreader_function(
spreader_variadic<1>(),
"Do something with X and REST\n\n(fn X &rest REST).",
[](envw env, value x, spreader_restargs rest) {
for (value val : rest) {
// ...
}
return nullptr;
});
env->*make_spreader_function(
spreader_variadic<1>(),
"Do something with X and REST\n\n(fn X &rest REST).",
[](envw env, value x, std::vector<value> rest) {
// ...
return nullptr;
});
#if defined(__cpp_lib_span)
env->*make_spreader_function(
spreader_variadic<1>(),
"Do something with X and REST\n\n(fn X &rest REST).",
[](envw env, value x, std::span<value> rest) {
// ...
return nullptr;
});
#endif
